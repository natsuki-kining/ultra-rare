# ultra-rare

#### 介绍
基于ssr跟datav实现的数据可视化大屏项目。
因为是基于ssr来写的，故取名为ur，即ultra rare。



# 大概功能

## 管理页面
* 大屏页面绘制
    * 展示模块
        * 数据刷新时间
        * 查询SQL
            * 动态生成SQL
        * 数据库选择
        * 样式选择 
        * 脚本编写与执行设置
    * 布局
    * 
* 大屏页面发布
    * 生成页面id与大屏页面相关数据
* 权限认证
    * 单点登录
    * 权限过滤    
## 大屏展示页面
* 根据id打开大屏页面
